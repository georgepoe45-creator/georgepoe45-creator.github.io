# Build Strong Apprenticeship America Website

This repository contains the public website for **Build Strong Apprenticeship America (BSA)**, a 501(c)(3) nonprofit based in Iroquois, Illinois. The site is a simple, static HTML page hosted on GitHub Pages.

## About BSA

Build Strong Apprenticeship America (BSA) provides fast‑track residential electrical and plumbing apprenticeships using mobile training sheds, retired field experts, and very small classes so rural residents can get job‑ready in six months. [cite:1][cite:3]

Our six‑month program mirrors first‑year apprenticeship skills in residential work, giving contractors “plug‑and‑play” apprentices and students a real path to a middle‑class income. [cite:1]
